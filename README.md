# Installation
If you want to run the latest code from git, here's how to get started:

1. Clone the code:

        git clone https://github.com/SINTEF-9012/SIS-Framework-Deployer.git
        cd SIS-Framework-Deployer

2. Install the dependencies

        npm install

4. Run

        npm start
